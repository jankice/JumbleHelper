Jumble Helper

This app helps you solve anagrams i.e. unjumble dictionary words.

For example, entering pplea will give apple

If mulitple words are found, the application displays all the words.

Clicking on each word will speak it, using in built Text To Speech engine.

Unix OS dictionary is used as a database of words.

Moreover, the application is ad-free.
